🔌 CPA 전환 추적 연동 가이드

CPA 광고를 집행하시나요? 전환(회원가입, 구매 등)이 발생했을 때 아래 API를 호출해주세요.

// POST Request
curl -X POST "/api/v1/track-conversion" \
-H "Content-Type: application/json" \
-d '{
  "log_id": "LOG_ID_FROM_URL_PARAM",
  "amount": 5000
}'

* 사용자가 광고를 클릭하여 귀사의 사이트에 도착할 때, URL 파라미터로 ?log_id=...가 전달됩니다.
* 이 log_id를 저장해두었다가, 전환 완료 시 API로 보내주세요.
